# CONSPOL

## Descrição

A CONSPOL é uma solução ao problema de comunicação entre partes. Com o objetivo de se encontrar o caminho com menor custo para dada comunição, se foi implementado o Algoritmo de Dijkstra.

## Sobre

A CONSPOL foi desenvolvida durante a disciplina de Teoria dos Grafos, no semestre 2019.1, ministrada pela Professora Patrícia Araújo de Oliveira.
